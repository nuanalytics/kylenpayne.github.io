We can also get a list of the means for each of these variables

## grab the variables that we made the boxplots with.
prots <- c('CREB_N', 'ERK_N', 'JNK_N') 
## grab those variables out of the data.frame and
## reassign them to a new data.frame
dc_prots <- data_cortex[,prots]

## colMeans calculates the mean for each column
colMeans(dc_prots)
##    CREB_N     ERK_N     JNK_N 
## 0.1731612 2.4823928 0.2350940
## standard deviation
## the FUN part mean 'function', i.e. which function
## you would like to apply to each column
sapply(dc_prots, FUN=sd )
##     CREB_N      ERK_N      JNK_N 
## 0.02563616 0.66446445 0.03234618
sapply(dc_prots, FUN=quantile)
##         CREB_N    ERK_N     JNK_N
## 0%   0.1136364 1.169514 0.1494872
## 25%  0.1530874 1.982083 0.2093339
## 50%  0.1724507 2.406569 0.2372972
## 75%  0.1906849 2.883884 0.2596802
## 100% 0.2560119 5.198404 0.3219850

sapply stands for ‘simple’ + ‘apply’, which means roughly, ‘apply this function to each column in my data.frame, then output the results in a simple format’. This is a nice function to use if you have to perform the same task over several variables. In fact the colMeans function is just a convinent wrapper for

sapply(dc_prots, FUN=mean)
##    CREB_N     ERK_N     JNK_N 
## 0.1731612 2.4823928 0.2350940
